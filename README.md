## Comandos

- mvn clean package

- scp -i ~ /<local chave .pem> target/pizzaweb-0.0.1-SNAPSHOT.jar <nome instância na aws>:~
  
- ssh -i ~/<local chave .pem> <nome instância na aws>
  
- java -jar  pizzaweb-0.0.1-SNAPSHOT.jar
  
- <nome instância na aws>:8080/pizzaweb/1
